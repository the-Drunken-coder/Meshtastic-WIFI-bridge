# Meshtastic WIFI Bridge

A bridge application for connecting Meshtastic devices over WiFi.

## Overview

This project provides a WiFi bridge for Meshtastic mesh networking devices, enabling communication between Meshtastic nodes over WiFi networks.

## Features

- WiFi connectivity for Meshtastic devices
- Bridge functionality between mesh networks
- Real-time message forwarding

## Requirements

- Python 3.x
- Meshtastic device
- WiFi connectivity

## Installation

```bash
# Clone the repository
git clone <repository-url>
cd "Meshtastic WIFI bridge"

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

## Configuration

Configure your settings in `config.py` or via environment variables.
